Difference between entity and event data

Event Data:

Sensors are infiltrating our lives, and businesses are trying to track every action a user takes. All this is done using event data. Just think about all smartphones, cameras, smartwatches, and other trackers that are constantly collecting data. But what are they collecting?

Event data mainly describes the actions performed by entities.
A Few key pieces of information that are captured in event data might be the action, timestamp and state.

Example:

The Action is when a customer opened a promotional email.

The Timestamp is when it happens - 6:15 pm October 6th.

The State will refer to all other relevant data such as device mobile, tablet, or computer and the browser than what browser or app was used, etc. An iPhone using the Gmail app.

Entity Data Event Data
Normalized Denormalized
Describes entities Describes actions of entities
Describes now Describes trends over time
Updates Appends
