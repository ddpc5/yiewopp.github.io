Additional Information about the Exercises

Each tab in the spreadsheet is correlated with an exercise in this course where it should be used. The document contains the questions relevant to each exercise.

Important: Keep a Copy of your Exercise Responses in the Template Document

It is highly recommended that as you complete the open-ended exercises in this course, you should keep them in the provided template document to refer back to throughout the course. It will be advantageous as you complete the Final Exercise in the course: Knapsack Data Strategy MVP in the Data Strategy section of this course. You can still copy and paste your responses into the text boxes on each page in the course for each exercise if you wish.

If you are using something other than Microsoft Office

This spreadsheet is saved in Excel format but can be opened in any spreadsheet application.

All video exercises were completed in Google Sheets/Docs, and if you want to follow along using Google Sheets/Docs or if you don't have Microsoft Office, please follow the direction below:

  • Upload the Exercises file into Google Drive
  • Click on it and choose Open With
  • Select Google Sheets/Docs depending on the file

If the file opens in Google Drive's Compatibility Mode, indicated by an Excel X or Word X icon in the top right of the spreadsheet,

  • Click on File in the Menu Bar
  • Click Save as Google Sheets/Docs
  • Google will convert the file and allow you to follow along using Google Sheets/Docs.

If you have other spreadsheet software or use MS Office, the Excel file should open in that software as well, but you will need to consult the documentation for that software to complete the activities in this course for your spreadsheet software as the steps may be different from Google Sheets/Docs.
