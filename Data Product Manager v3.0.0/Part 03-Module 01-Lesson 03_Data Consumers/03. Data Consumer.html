Understanding Data Consumer Better

Getting data is excellent, but being able to use it is very important. To do that, we need to understand who a data consumer is? A data consumer would be a person, an organization, or a system that uses data for a specific purpose. A data consumer can also be a data producer. How is that possible?

To further understand our data stakeholders, we will have to dive deep into:

  • How does a data consumer want to use data? Is it for reporting, creating a visualization, for reconciliation, or building ML models?
  • What tools are being used by users? Microsoft Excel, Tableau, Microsoft Power BI, or Google Data Studio.
  • Are there any standards within a particular community of users? Legal might want data differently than accounting or finance.

Here we also need to understand that sometimes we might have to provide the same data in multiple formats to satisfy the needs of different consumer groups.
