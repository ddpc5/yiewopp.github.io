Solution: BuyStuff.com New Users

ND030 C3 L3 08 Identifying Behavioral Exercise Solution Video

Summary

Question 1: List all the user characteristics or behavioral segments available in the data set.

By uploading the raw file into Tableau, we can quickly review the column names and the different values within each column. By placing a dimension from the table into a column or row in Tableau, the number of unique names per dimension will be shown. In the case of this data file, the columns with bearing on user characteristics/behavior include Sex, Device, and Date.

Question 2: List the divisions for each demographic or behavioral segment in the data set.

The gender segments are male and female.

The device segments are desktop and mobile.

The dates, which might refer to the date the user was first seen, including dates from January, February, March, and April in the year 2015.

Question 3: Which is the largest user segment group? Which is the smallest user segment group?

The largest segment group is desktop users, who represent 60,200 users. The smallest segment group is the monthly number of records, at 22,600 records per month. For user traits, the smallest segment group is mobile users, with 30,200 mobile records.
