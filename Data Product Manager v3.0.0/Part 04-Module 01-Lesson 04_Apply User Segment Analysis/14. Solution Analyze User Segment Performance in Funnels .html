Analyze User Segment Performance in Funnels Solution

ND030 C3 L3 11 Applying Cohort Analysis To Funnel Analyses In Tableau Exercise Solution Video

Summary

Begin just as you did when calculating the overall drop off rate between pages on BuyStuff.com. To see the drop off between pages, first count the number of distinct events on each page. Because this website has a simple flow, every page is required of the user to complete a purchase. Therefore, every page will be shown in the tunnel and every page represents a user taking new action. To calculate the actual drop off, use a Quick Table Calculation of percent change between counts.

1) Determine the drop off rate at each step in the BuyThings.com e-commerce funnel

Add the behavioral aspect to a row or column view to determine the drop off rate for a given behavior/characteristic. By looking at device, sex, and month separately, you'll be able to see where the drop off rates vary among cohorts. See the images below.

2) Determine and write findings of cohort analysis for devices, sexes, and months. Which cohorts that have a higher average drop off rate between steps than other users and what could such a drop off rate indicate?

Sex: For males and females, there is no difference in drop off between steps. This indicates gender may not play a role in whether a new user completes the shopping flow.

Device: Mobile users have a 10% higher drop off rate between the search and payment pages (90% conversion on desktop versus 80% on mobile). They also have a 5% higher drop off rate between the payment and payment confirmation pages. These may indicate areas for mobile experience optimization.

Month: A large increase in drop off between landing and search pages can be seen when comparing January/February to March/April. Drop off increases from about 40% to about 60% between the first two pages. Similarly, the drop off rate between search and payment pages increases 10% from around 82% to around 92%. This indicates a change occurred between February and March that is impacting the new user funnel conversion. The CEO should examine what changed from February to March. Perhaps a new feature introduced a bug or a new marketing method is recruiting more users who are less likely to purchase.
