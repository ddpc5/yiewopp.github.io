02. Big Picture / Intuition

Big Picture / Intuition

ND030 C3 L3 02 Big Picture

Summary

User segment analysis allows product managers to look at data from the perspective of different types of customers.

User segments are groups of users that share a similar characteristic. A user segment can be based on dates a user started a product, user demographics, and behavioral characteristics.

Users are not all the same, and segment analysis allows us to find patterns in the data about how our product is succeeding or failing specific types of users. The findings from segment analysis often drive further research that leads to improvement enabling the product to better appeal to user segments of interest.

Which of the following are true about user segment analysis? (select all that apply)
SOLUTION:
  • User segments are groups of users that share a similar characteristic
  • User segments can be based on when the demographics of a user
  • User segments can be based on the behavior of the users
  • User segment analysis helps product managers find places where a product or feature works better for some than others
