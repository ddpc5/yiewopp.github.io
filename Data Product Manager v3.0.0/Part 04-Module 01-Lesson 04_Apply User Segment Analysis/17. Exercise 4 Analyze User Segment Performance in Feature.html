17. Exercise 4: Analyze User Segment Performance in Feature

Analyze User Segment Performance in Feature Usage Exercise: Random Song Feature Analysis

Instruction

A music app allows users to search for and play a specific song. In the second week of October of 2019, they launched a new feature that allowed first-time visitors to play a random song. In this exercise, you will perform a feature analysis to determine the popularity of the feature with new users. Also, break the users into segments and confirm if both features were used at roughly the same rate among different user segments, or if one feature appealed more to a certain user group.

The data that they provided contains information of first-time users only. However, the users could use the features multiple times.

Click here to download data.

  1. Calculate the number of distinct users that used each feature at least once
  2. Based on available segments, analyze the data to determine if the use of the two features was different among segments. Explain your answer using numbers as evidence of these findings.

Hint: COUNT userIDs will count all the uses of each feature. COUNT (distinct) userIDs will count unique users who used each feature at least once.
