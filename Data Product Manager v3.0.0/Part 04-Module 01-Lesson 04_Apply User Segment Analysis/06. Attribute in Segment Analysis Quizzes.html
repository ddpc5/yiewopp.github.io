Cohort Analysis

The cohort analysis graph above represents the percent of visitors to a website that made a purchase X number of days after their first visit to the site. Based on this chart, which starting date segment was most likely to make a purchase on the day after they first arrived at the website? (Hint: Day 0 is the first day a user visited the website, Day 1 is the first day after the first visit, etc.)
SOLUTION: October 14

Which of the following could be used for demographic segmentation? (Select all that apply)

  • User age

Behavioral Segmentation

Which of the following could be used for behavioral segmentation? (Select all that apply)
SOLUTION:
  • Number of logins per day
  • User has or has not used a specific feature
