Lesson Conclusion

Summary

Congratulations! You have just finished the fourth lesson of this course. In this lesson, we learned how to apply user segment analysis to reveal opportunities to optimize products for different types of users.

First, we discussed different types of user segments and how to break down users into smaller groups based on their start date, demographics, and behavior.

Second, we learned how to apply segment analysis to funnel analyses in Tableau.

Finally, we learned how to use segment analysis when reviewing feature usage data.

Segment analysis is an important part of the data-driven iterative design cycle. By adding segment analysis to funnel and feature analyses, product managers can further identify how different types of users use a product and can identify areas for improvement to better serve different segments of users.

Glossary

  • Cohort: For analysis purposes, cohorts refer to groups of people who begin using a product or feature at the same time.
  • Cohort Analysis: A type of segment analysis that focuses specifically on user cohorts.
  • Demographics: Anything that describes a user as a person, such as income, age, and gender.
  • Segment Analysis: The process of analyzing how different groups of users compare in product or feature usage.
