03. Lesson Outline

Lesson Outline

ND030 C3 L3 03 Lesson Outline Video

Summary

In this lesson, we’ll first learn how to segment user data and discuss how those types of user behaviors and characteristics can be used in segment analysis.

In the second part of the lesson, we’ll learn how to apply segment analysis to user conversion funnels. We’ll discuss what data patterns provide evidence that segments are performing differently and when further research would be a good next step.

Finally, in the third part of the lesson, we’ll learn how to apply user segment analysis to feature usage. As with funnel analysis, we’ll discuss what data patterns provide evidence that different types of users are performing differently. We’ll learn when further research at a user segment level would be a good next step for improving a feature.

Lesson objective

By the end of this lesson, you will be able to

  • Explain the importance of segmenting user data
  • Identify behavioral traits in a data set that could be used to analyze user behavior
  • Apply segments to funnel analysis
  • Calculate feature usage within a product, both among all users and among selected segment groups using existing event data
