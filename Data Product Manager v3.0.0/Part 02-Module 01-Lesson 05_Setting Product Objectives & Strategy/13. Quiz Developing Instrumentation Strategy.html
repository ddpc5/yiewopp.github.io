13. Quiz: Developing Instrumentation Strategy

Instrumentation Strategy Quiz 1

QUIZ QUESTION::

Let’s say you’re a PM at an online video hosting company. Match the below user actions to the event names they are most likely associated with.

ANSWER CHOICES:



User Action

Event Name

User plays a video.

User enters the homepage.

User pauses a video.

A video that a user was watching has ended, either through browser exit, clicking on another video, or the video finishes playing.

User subscribes to a creator.

User views an ad.

User posts a public comment on a video.
SOLUTION:

User Action

Event Name

User pauses a video.

User subscribes to a creator.

User plays a video.

User enters the homepage.

User views an ad.

A video that a user was watching has ended, either through browser exit, clicking on another video, or the video finishes playing.

User posts a public comment on a video.

Pick all the event properties that make sense to be associated with the event of videoEnded, that fires when a video that was being watched is ended, for whatever reason.

SOLUTION:
  • method_of_exit (potential values: browser_close, video_completed, different_video_played)
  • total_time_watched
  • video_id
  • user_id
