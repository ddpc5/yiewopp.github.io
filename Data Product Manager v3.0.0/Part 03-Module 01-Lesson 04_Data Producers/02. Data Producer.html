Who is Data Producer?

Data Producer: A system, service, or user that produces data relevant to a business.

A data consumer can be a data producer. Consider a relational database storing online transactions. It will be a valuable source of shopping trends and patterns for the marketing team.

Data Producer Quiz

SOLUTION:
  • Websites can be data producers
  • Soil moisture sensor can be a data producer
  • Data user can be data producer
