Event and Event Data

In today's world of connected devices and social media, it is estimated that 2.5 exabytes of data is produced every day, which is equal to 90 years of HD video.

A considerable portion of this data is event data. Therefore, event data is essential, it will influence how successfully we navigate data product management.

An Event is an interaction or an action at a particular moment by a user, a system, or a sensor.

Event Data is generated by such events
Example:
Website - User Interaction

  • Downloads
  • Link clicks
  • Form submission

Social Media

  • Shares
  • Likes
  • Mention
  • Hashtags

Systems

  • Data Replication
  • Data Syncing
  • Deletion of jobs

Sensors

  • Motion Detection
  • Voice Detection
  • Smoke Detection
  • Temperature Detection

Major Event Data Components:

  • Action
  • Timestamp
  • State

Event Data Quiz 1

What is event data?
SOLUTION:
  • Data that gets generated by user interaction on website
  • Data that is generated by smart devices in homes

Event Data Quiz 2

Which of the following is not event data?
SOLUTION:
  • Data you query using SQL
  • Data stored in a DB after an online transaction is completed

Event Quiz 3

Which of the following is part of an event data?
SOLUTION:
  • Type of interaction the user had with the app - opened it, closed it, downloaded something, searched something
  • When exactly did the user do all of this?
  • What device was the user using?
