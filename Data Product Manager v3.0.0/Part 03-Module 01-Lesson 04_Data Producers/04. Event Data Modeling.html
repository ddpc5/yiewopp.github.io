Performance Indicator

Ways to measure KPIs

Number of users actively using your website or App. Generally measured as :

  • Daily Active Users (DAU)
  • Weekly Active Users (WAU)
  • Monthly Active Users (MAU)
  • Primary Measure of User Growth

Session Length

  • Time spent by a user using your app in one session
  • The measure of User Engagement

Click-Through Rate

  • The ratio of users who clicked an ad or banner versus total users who viewed the page with that ad or banner
  • Measures Users Interest in Content or Content Relevance

Bounce Rate

  • Percentage of users leaving the website after viewing only one page
  • Measures effectiveness of entry page

Conversion Rate

  • Most important KPI
  • Percentage of users completing the desired action.
  • Desired actions
    • Buying an Item
    • Completing Registration
    • Downloading File
  • Measures Interest of User in Offering
