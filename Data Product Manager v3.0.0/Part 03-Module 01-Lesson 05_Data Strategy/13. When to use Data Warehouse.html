13. When to use Data Warehouse

NDO30 DPMND C2 L4 12 When To Use Data Warehouse Heading

When Data Warehouse

When to use Data Warehouse

  • Multiple Database across the organization
  • Large Database
  • Lots of historical data.
  • DWHs are best for data mining in such scenarios.
  • Market research analysis. When large volumes of data are needed to do in-depth analysis with multiple dimensions of data help
  • User behavior analysis needs a large volume of data, and data is needed from multiple sources.

DW Quiz

When should you use a Data Warehouse…
SOLUTION:
  • When data needs to be combined from various Database
  • When large volumes of data needs to be analyzed
  • When data needs to be analyzed on multiple dimensions
  • When you need to process huge historical data
