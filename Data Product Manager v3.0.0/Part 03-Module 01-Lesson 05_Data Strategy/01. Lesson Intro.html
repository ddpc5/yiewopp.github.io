Lesson Outline Recap

In this lesson we will cover:

  • Processing of data with ETL versus ELT
  • Data storage options
    • How to choose the best option between a Database, a Data Warehouse or Data Lake
  • Data strategy
    • When should you build and when should you buy?
    • Open source vs commercial
    • Cloud Vs. On-prem.
    • Insource Vs. outsource the development of tech.
  • Security and compliance.
