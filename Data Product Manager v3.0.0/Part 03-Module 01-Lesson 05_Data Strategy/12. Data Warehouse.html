Data Warehouse

Quick Reminder:

A data warehouse is a central repository of information that can be analyzed to make better-informed decisions. Data flows into a data warehouse from transactional systems, relational databases, and other sources, typically on a regular cadence.

Advantages

  • Data consolidated at one place
  • Speed
  • Enriched data -> Better Business Intelligence
  • Better decision making
  • Improved Data Quality

Disadvantages/Limitations

  • Could be Costly
  • Requires continuous cleaning, transformation and data integration
  • Not helpful for real-time decision making
  • Once implemented adding a new data source might be challenging
