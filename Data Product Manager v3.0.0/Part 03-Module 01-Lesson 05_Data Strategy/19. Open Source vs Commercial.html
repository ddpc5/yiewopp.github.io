Open Source

Open Source vs Commercial

Open-source software (OSS) is a type of computer software in which source code is released under a license in which the copyright holder grants users the rights to study, change, and distribute the software to anyone and for any purpose.

When to use Open Source

Use Open Source:

  • If someone has already created what you need.
  • When the code is well supported by its community.
  • When Total Cost of Ownership (TCO) is low
  • When it aligns with what your brand stands for
  • When you need source code not just an application

Commercial

Use commercial if

Commercial Software is a software that is built by a person or group to be sold at a price. And depending on the software and price, source code may or may not be included.

When to use Commercial Software

Use commercial software if:

  • Total Cost of Ownership (TOC) is high for open source
  • Security is a big concern
  • It is easier to adapt to your needs
  • it is an Industry standard solution like spreadsheets everybody uses it
  • You need reliable support

OS vs Commercial Quiz

Which of the following is false?
SOLUTION:
  • You should not use the most popular commercial softwares because your competitor might also be using the same.
  • Support is not important because you can always train people to support functionalities and they can always solve problems for you.
  • You never need source code for the software you buy.
