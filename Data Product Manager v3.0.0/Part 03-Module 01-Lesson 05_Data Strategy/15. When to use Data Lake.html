When to use a Data Lake

  • Experimental Analysis

Data lakes can be used for experimental analysis of data, where you are still not sure and want to have a proof of concept before investing in an end-to-end data pipeline. ELTs make it easy to use Data lakes for such experimental analysis.

  • Cybersecurity

Cybersecurity data collection and analysis is a great use case of data lakes. Any enterprise devices such as laptops, servers, cloud computing devices, and virtual systems all are vulnerable. Add connected devices to this mix. It becomes a big data problem that can be solved using Data lakes.

  • Customer Data Platform

Data Lakes can be used to build customer data platforms to provide rich customer insights. Data for such platforms can come from various sources such as

  • customer profile Databases
  • web and mobile events
  • brick and mortar systems
  • loyalty systems
  • service center data

This data can be of any type of structured, semi-structured, or unstructured. Data Lakes work best for such use cases.

DL Quiz 2

When should you use Data Lake
SOLUTION:
  • If it is not completely clear what data needs to be used.
  • When data needs to be collected from very many sources, and is of all types.
  • When ELT is the best processing strategy
  • When you have big data problems
