Data Lakes

A data lake is a system or repository of data stored in its natural/raw format,usually object blobs or files.

Broken down the definition of what data lake is:

  • A Repository of Data that Stores data in its natural form
  • Usually a single store of all enterprise data
  • Used for reporting, visualization, advanced analytics, and machine learning
  • Has structured, semi-structured and unstructured data

Advantages

  • A vast amount of data can be stored.
  • It can be
    • Structured
    • Semi-structured
    • Unstructured data
  • Processing before loading not required
  • Highly Agile

Disadvantages/Limitations

  • Maturing Technology
  • Data Governance
  • Low-cost storage might encourage unnecessary data accumulation
  • Privacy Issues
