Challenges with ETL

ETL Challenges

Some of the challenges with ETL include:

  • Processing data before loading it.
  • Amount of data generated in an enterprise is increasing
  • Transformation code can be millions of lines. This might make it really hard to scale an ETL pipeline.
  • Transforming all the data before loading might be an overkill.
    • Example: you might not need to process all the event data generated on the website at once.

Challenges continued:

Different Data Types

Entity data needs to be treated very differently than event data. Similarly, structured data, semi-structured data, or unstructured data. All have specific needs. ETLs work best with entity and structured data. Implementing ETL for other data types can be challenging.

Different Data Sources

Data sources are varied and spread across. We have data on-prem and in the cloud. We have constant streams of data, and we have batches of data. With ETL pipelines it might be difficult and slow to bring all this data together.

ETL Quiz 1
