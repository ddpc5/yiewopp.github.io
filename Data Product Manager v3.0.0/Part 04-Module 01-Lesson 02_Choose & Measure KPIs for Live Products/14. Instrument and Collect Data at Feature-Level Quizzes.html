Instrument and Collect Data at Feature-Level Quizzes

Feature Instrumentation

Which of the following are important to do when instrumenting a new feature? (select all that apply)
SOLUTION:
  • Track separate events for separate user actions.
  • Define and document any new events you create when instrumenting
  • Track both event-level data and session-based data such as a user's browser or device type

Feature Instrumentation

Scenario: A data product manager is launching a feature in a map app to give more value to customers by including the history of buildings while a user is walking. Users can click on a "learn more" icon about the buildings they are interested in. From the choices below, select all instrumentation that would be important to put in place for the new feature.
SOLUTION:
  • How many users walked past buildings of interest
  • Which users selected the "learn more" option
  • How many times the average user chose to learn more during a walking session
