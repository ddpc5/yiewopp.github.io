Alex Pipeline Components

Data Processing: the first component is Amazon S3, Amazon Lambda

Lambda will place data into an S3 bucket again, here if you notice our S3 buckets are intermediate storages.

From Amazon S3, data is pulled by Amazon EMR, which is a Big Data Platform service. The data is used for distributed storage, and to run compute over big data, it uses frameworks such as Hadoop and Apache Spark.

Data Storage: we already know for intermediate data storage we are using S3, along with that, we can use Amazon Redshift which is a data warehouse.

Data Destinations: Amazon Sagemaker a cloud machine-learning platform and Amazon Quicksight a business intelligence service
