Introduction

Summary

Working with data can be overwhelming: which data is the most important? Which sources should you focus on? How can the data be leveraged to improve your product? By the end of this course, you’ll have the skills to answer these questions and to leverage data in a more effective way.

In data-driven iterative design, data product managers evaluate user data to identify areas for improvement, then look for potential solutions, create prototypes, get feedback from customers, launch tests, and then return to data evaluation to look for more areas to improve and grow user satisfaction. By the end of this course, you will learn about the iterative design process and how to use both quantitative and qualitative data to help improve your products and better serve your customers.
