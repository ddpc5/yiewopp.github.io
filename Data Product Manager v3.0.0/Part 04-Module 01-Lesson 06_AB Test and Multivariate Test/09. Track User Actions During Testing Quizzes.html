09. Track User Actions During Testing Quizzes

Track User Actions During Testing Quizzes

As a data product manager, you are testing a new feature that allows users to change the product website color scheme to "dark mode." Which of the below metrics would be important to track when running the test? (Select all that apply)
SOLUTION:
  • The time a user uses the feature
  • The device each user is on
  • The percent of users who used the dark mode feature
  • The conversion rate (those who see the feature and use it)
  • Which users are exposed to the dark mode feature

Tracking User Test State Exposure

Why is it important to track to which test state a user was assigned?
SOLUTION: To be able to calculate the conversion rate per test state
