Solution: Calculate & Respond to Test Results Exercise

ND030 C3 L5 13 Calculating & Responding To Test Results Exercise Solution Video

Summary

1) Calculate the number of users in the test and control group

To calculate the number of users in the test group, count the number of unique user IDs associated with the new landing page. There are 146,317 unique users in the test group and 146,265 unique users in the control group. If you got 147,239 users remember that you need to count the number of unique users, not the number of times any person visited the new test landing or old control page.

2) Calculate the number of conversions in the test and control group

By counting the number of "converted = 1" when landing page = "new_page" or landing page = "old_page" , you will know the number of conversions (i.e. the number of times users subscribed to the newsletter) in the test and control groups. For the test group, there were 17,498 conversions. For the control group, there were 17,739 conversions.

3) Based on the the numbers produced in question 1 and 2, using a 2-sided T test and the statistical significance calculator of your choice, are the test results significant at a 95% confidence level? How do you know?

Using a two-sided T test at a 95% confidence interval finds the results of the test are not significant. We know the results are not significant because the p-value generated is .9199 which is larger than the p-value of .025. The large p-value indicates that the difference in conversion between the control and test group is not significant at a 95% confidence level in a two-sided T test. An image of the p-value calculation from an online statistical significance calculator is included below for review.

4) Should the landing page be rolled out to all users? Why or why not?

The landing page should not necessarily be rolled out to all users as the results of the test are not significant. When results are not significant that means the test condition did not result in a statistically significant increase in conversions, which means any difference between the two landing pages is more likely due to random chance than to the differences between the two pages. This means the new test landing page cannot be said with confidence to lead more users to sign up for the newsletter. The website should continue to test different landing pages to see if they can generate a significant test result that shows higher conversion in the test condition.
