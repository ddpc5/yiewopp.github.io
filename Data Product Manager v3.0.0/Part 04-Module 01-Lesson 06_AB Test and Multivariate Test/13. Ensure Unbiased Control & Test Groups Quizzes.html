Ensure Unbiased Control & Test Groups Quizzes

Balancing User Types Among Control and Test Groups

Why is it important to achieve balance among user types in the test and control groups?
SOLUTION: To ensure that any difference in user conversion rate is not from a difference between the users, but caused by a different between the test and control states

If you randomize the users placed into the "A" (control) or "B" (test) group, which of the following user characteristics would you expect to be balanced between the groups?
SOLUTION:
  • The percent of users in each group who are male
  • The number of users in the control and test group
  • The percent of users in each group who use the product daily
