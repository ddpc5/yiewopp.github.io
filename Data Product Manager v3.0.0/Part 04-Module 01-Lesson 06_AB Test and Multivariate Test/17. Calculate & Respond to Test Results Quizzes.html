17. Calculate & Respond to Test Results Quizzes

Calculate & Respond to Test Results Quizzes

Select all the statements that are true about T-Tests as used in product management when evaluating the results of an A/B test or comparing the results of different multivariate test states.
SOLUTION:
  • Most product managers use two-tailed T tests as their experiments can make conversion rates go up and down
  • Most product managers look for a p-value below .025 to indicate statistical significance of test results

Read the test results for a new landing page, should the product manager expand the test state to all users? You may use an online A/B Testing calculator.

5,000 users in the control state and 5,000 users in the test state
400 first time visitors in control state converted from visitor to registered user
500 first time visitors in the test state converted from visitor to registered user
SOLUTION: Yes, the product manager has statistically significant results with a p-value below .025; the new landing page will likely increase conversion

Read the test results for a new landing page, should the product manager expand the test state to all users?

8,000 users in the control state and 8,000 users in the test state
1,500 first time visitors in control state converted from visitor to registered user
500 first time visitors in the test state converted from visitor to registered user
SOLUTION: No, the results are statistically significant, but the conversion rate is lower in the test state so the test state should not be expanded to all users.
