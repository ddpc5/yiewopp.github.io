18. Exercise 4: Calculate & Respond to Test Results

Exercise 4: Calculate & Respond to Test Results

Instruction

A website launched a classic A/B test of its landing page. For about half the users, the original landing page was shown. For about half the users, a new design was shown but still had the same call to action, to subscribe to a newsletter.

The data that they provided contains information from the A/B test. In the data, you'll find a column with user IDs. The timestamp column represents when the user landed on the home page. You'll also find a "page" column that shows whether the user was shown the new, test page ("new_page") or the old, control page ("old_page"). The column "converted" shows whether a user subscribed to the newsletter ("1") or did not subscribe to the newsletter ("0").

Analyze the data and answer the following questions. Click here to download data.

  1. Calculate the number of users in the test and control group
  2. Calculate the number of conversions in the test and control group
  3. Based on the numbers produced in question 1 and 2, using a 2-sided T-test and the statistical significance calculator of your choice, are the test results significant at a 95% confidence level? How do you know?
  4. Should the landing page be rolled out to all users? Why or why not?
