Visualize Funnel Drop Off in Tableau Exercise Solution: Ecommerce Funnel

ND030 C3 L2 11 Visualizing Funnel Dropoff In Tableau Exercise Solution Video

Summary

As with the exercise in the section on calculating funnel dropoff, to determine the funnel drop off, you must first confirm the order of pages and/or user activity in the funnel of interest. Under the "page" column in the BuyStuff.com data set, you will see four different pages: home page, search page, payment page, and payment confirmation page. These four pages align with an e-commerce sales funnel. Your website's visitors must land on a landing page ("home page") and then look for an item to purchase ("search page") before going to purchase the item ("payment page") and finally submitting their payment information ("payment confirmation page"). Make sure to order the pages in Tableau in the same order.

Whether calculating or visualizing, first count the number of distinct events on each page. Because this website has a simple flow, every page is required of the user to complete a purchase. Therefore, every page will be shown in the tunnel and every page represents a user taking new action.

A bar chart is a clean and simple way to show drop-off between steps in a funnel. While there are more "funnel" like visualizations possible, the following instructions confirm how to create a bar chart from the data:

  1. Upload the data into Tableau. Put the page events into the order they occur in the sales funnel and count the number of distinct events on each page.

  2. Select the bar chart from the "Show Me" dropdown to transform the data. See the image below.

  1. Select "Labels" from the side navigation and check "Show mark labels." This will cause the number of visitors to each page to be displayed above each bar. Such labeling often makes it easier for viewers to understand the raw numbers associated with a visual.

  1. Expand the chart large enough that all labels are readable. This will make your chart ready for the board room!
