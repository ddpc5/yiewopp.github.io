Visualize Funnel Drop Off in Tableau

ND030 C3 L2 10 Visualizing Funnel Dropoff In Tableau Walkthrough

Click here to download the data.

Summary

To visualize the funnel drop off in Tableau, you start with review what data is available, putting steps in order, and counting the number of users who make it through each step. Then you create a visualization.

The simplest visualization is to create a bar chart. To create a bar chart, you can select that visual type from the visual section.

Before the visual can be put in front of others, you need to make sure all the labels are readable and correct.

Next, you need to make sure the title clearly represents what is being visualized.

While the bars show the relative drop at each step, it is always nice to provide the exact numbers for easy reference. To do this, you will go to the marks section and add labels

Finally, before sharing a visual, it is helpful to distinguish the steps with different colors. This makes them stand out as separate moments in the user journey. To color the bars, you need to drag the dimension of interest over the color section in marks.
