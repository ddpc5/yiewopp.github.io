03. Lesson Outline

Lesson Outline

ND030 C3 L2 03 Lesson Outline Video

Summary

In this lesson, we’ll first learn about the steps users take towards a goal and form a funnel and then learn to apply the same technique to feature usage.

In the second part, we’ll discuss how to calculate funnel drop off in Tableau and identify the steps in a user funnel with the highest drop off rates.

Finally, in the third part of this lesson, we’ll discuss how to visualize funnel drop off in Tableau for use in presentations.

Lesson objective

By the end of this lesson, you will be able to

  • Identify the steps in typical user acquisition and activation funnel
  • Run analysis in Tableau to determine the rate of user dropoff during each step of a funnel
  • Visualize a funnel analysis in Tableau in bar chart form
