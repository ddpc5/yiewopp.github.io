02. Big Picture / Intuition

Big Picture / Intuition

ND030 C3 L2 02 Big Picture Video

Summary

User funnel analysis tracks the step-by-step use of a product or a feature.

After identifying each step users take, product managers can analyze user data and calculate which steps are prone to drop off. Steps with a large dropoff are usually areas that need further investigation and improvement.

Which of the following is true about user conversion funnels?
SOLUTION:
  • Track users along a set of actions towards a goal
  • Indicate specific steps in a process which are prone to user dropoff
