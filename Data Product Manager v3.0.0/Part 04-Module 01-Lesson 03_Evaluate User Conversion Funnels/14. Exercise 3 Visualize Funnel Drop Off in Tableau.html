Instruction

Using the same dataset from the previous exercise, transform the BuyStuff.com funnel into a bar graph using Tableau. The bar chart shows the number of users who accessed each page. Be sure to title the graph and ensure all the labels can be read so that the board members know what the bar chart represents.
