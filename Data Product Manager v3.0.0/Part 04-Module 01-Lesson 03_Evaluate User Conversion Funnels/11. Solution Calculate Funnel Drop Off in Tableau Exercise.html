Calculate Funnel Drop Off in Tableau Exercise Solution: Ecommerce Flow

ND030 C3 L2 09 Calculating Funnel Drop Off In Tableau Exercise Solution Video

Summary

To determine the funnel drop off, first, upload the data file in Tableau. Under the "page" column, you will see four different pages: home page, search page, payment page, and payment confirmation page. These four pages align with an e-commerce sales funnel. Your website's visitors must land on a landing page ("home page") and then look for an item to purchase ("search page") before going to purchase the item ("payment page") and finally submitting their payment information ("payment confirmation page"). Make sure to order the pages in Tableau in the same order.

To see the drop off between pages, first count the number of distinct events on each page. Because this website has a simple flow, every page is required of the user to complete a purchase. Therefore, every page will be shown in the tunnel and every page represents a user taking a new action. To calculate the actual dropoff, use a Quick Table Calculation of percent change between counts. See image below.

1) List the percent change (drop off) between each page/activity.

See the example chart below! Note the chart is relabelled as "Conversion Tunnel Drop off Rate" to ensure it is clear to viewers the purpose and meaning behind the numbers below.

Based on this chart we can see a 50% drop off between home and search page, a 86.6% drop off between search and payment pages, and a 92.5% drop off between the payment and payment confirmation pages.

2) Between which pages/actions do users drop off the least?

Users drop off the least between the home_page and the search page. About 50% of all users continue from home page to search page.

3) Between which pages/actions do users drop off the most?

Users drop off the most between payment page and payment confirmation page, with 92.5% of users still in the funnel dropping out at this time. You might suggest to your CEO that you talk with users and investigate why this drop occurs. Perhaps the checkout page is difficult to use. Or perhaps there is another disincentive for customers to continue, such as a surprisingly high shipping fee.
