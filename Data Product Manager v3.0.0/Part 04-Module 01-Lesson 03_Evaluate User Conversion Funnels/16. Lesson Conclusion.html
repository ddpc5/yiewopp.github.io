16. Lesson Conclusion

Lesson Conclusion

ND030 C3 L2 12 Lesson Conclusion

Summary

Congratulations! You have just finished the third lesson of this course on evaluating user funnels to reveal opportunities for product improvement.

In this lesson, we first learned about how to create a user funnel. We examined the stages of the traditional sales funnel, from awareness to evaluation to purchase, and how to break down a user’s actions as steps towards a goal. Then we learned how these funnel analysis techniques could be applied to feature usage.

In the second part, we learned how to calculate funnel drop off in Tableau. You need to evaluate the available data, order events as a user would experience them, count unique sessions in each step, and finally calculate the percent change between steps.

Finally, in the third part, we learned how to visualize the user funnel in Tableau using a bar chart. Converting raw data into appealing visuals will strengthen our case in front of stakeholders.

Glossary

  • Awareness: The step in a sales funnel when the user becomes aware of a need and/or a product.
  • Evaluation: The step in a sales funnel when the user researches their options and considers whether a product will bring them value.
  • Purchase: The step in a sales funnel when the user takes the final/desired action of becoming a paying customer
