Narrow Scope with the RICE Framework Exercise Solution

ND030 C3 L4 13 Narrowing Scope With The RICE Framework Exercise Solution Video

Summary

1) Using the RICE framework, what is the score of each of the 5 features?

To calculate the RICE score for each feature, multiple the REACH, IMPACT, and CONFIDENCE together for the feature, then divide this total by the EFFORT to get the final RICE calculation.

  • Ability to share individual pages via a quick link: score of 600
  • Create line, bar, and pie charts based on collected data: score of 810
  • Single Sign-On with Social Media Sites (Google): score of 266.67
  • Single Sign-On with Social Media Site (Facebook): score of 50
  • Uploading external files to attach as notes: score 712.5

2) In what order should the 5 features be prioritized according to the RICE method?

Using the rice method, the highest scored features should be prioritized first. Thus, using the calculated scores, the features should be built in the following order:

  1. Create line, bar, and pie charts based on collected data
  2. Uploading external files to attach as notes
  3. Ability to share individual pages via a quick link
  4. Single Sign-On with Social Media Sites (Google)
  5. Single Sign-On with Social Media Site (Facebook)
