RICE Framework Calculations

Calculate the RICE score for the following features and identify which should be built first according to the RICE framework.

  • Feature 1: Reaches 100 users, medium impact (score of 2), confidence in estimates high (.9), effort level 2 weeks
  • Feature 2: Reaches 50 users, large impact (score of 3), confidence in estimates high (.9), effort level 3 weeks
SOLUTION: Feature 1 score = 90, Feature 2 score = 45. Feature 1 has the highest score and should be built first
